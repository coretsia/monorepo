# Coretsia Website Architecture

Status: Draft

This document will define the accepted website direction for Coretsia after the public design RFC is completed.

Current RFC:

- [Website design RFC](https://github.com/coretsia/monorepo/discussions/51)

The website direction must remain aligned with:

- [Branding specification](./BRANDING.md)

## Scope

The initial website direction is expected to cover:

- Homepage
- Documentation section
- News/blog section
- Community/contribution section
- Navigation model
- Light/dark visual treatment
- Brand application rules

## Status

No final website design direction has been accepted yet.

This document will be updated after the RFC discussion produces a clear direction.
